Тема: Программирование ESP8266 и создание веб-сервера

🎯 Цель урока

Научиться программировать микроконтроллер ESP8266 и создать собственный веб-сервер для управления устройствами через интернет.

🧠 Что такое ESP8266?

ESP8266 — это недорогой микроконтроллер с встроенным Wi-Fi. Он позволяет создавать интернет-вещи (IoT) с минимальными затратами.

  • Процессор: 32-битный, 80 МГц
  • Память: 4 МБ Flash
  • Wi-Fi: 802.11 b/g/n
  • Порты: GPIO, UART, I2C, SPI
  • Цена: ~200-500 рублей
⚠️ Важно: ESP8266 работает с напряжением 3.3V, а не 5V! Не подключайте напрямую к 5V — это сожжёт микроконтроллер.

🔧 Установка Arduino IDE для ESP8266

1 Установите Arduino IDE

Скачайте с официального сайта: arduino.cc

2 Добавьте платформу ESP8266

Откройте Файл → Настройки и в поле Дополнительные ссылки для менеджера плат вставьте:

https://arduino.esp8266.com/stable/package_esp8266com_index.json

3 Установите пакет ESP8266

Перейдите в Инструменты → Плата → Менеджер плат, найдите esp8266 и установите.

4 Подключите ESP8266

Подключите ESP8266 к компьютеру через USB-кабель. Выберите плату: Инструменты → Плата → ESP8266 → NodeMCU 1.0 (или вашу модель).

Выберите порт: Инструменты → Порт (например, COM3 на Windows или /dev/ttyUSB0 на Linux).

💻 Создание веб-сервера

Вот базовый код для создания веб-сервера на ESP8266:

// Подключаем библиотеки #include <ESP8266WiFi.h> #include <ESP8266WebServer.h> // Настройки Wi-Fi (ЗАМЕНИ НА СВОИ!) const char* ssid = "Ваш_WiFi_SSID"; const char* password = "Ваш_пароль_WiFi"; // Создаём сервер на порту 80 ESP8266WebServer server(80); void setup() { Serial.begin(115200); WiFi.begin(ssid, password); // Ждём подключения while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println("Подключено!"); Serial.print("IP-адрес: "); Serial.println(WiFi.localIP()); // Обработчик для главной страницы server.on("/", handleRoot); server.on("/on", handleOn); server.on("/off", handleOff); server.begin(); } void loop() { server.handleClient(); } // Страница с управлением светодиодом void handleRoot() { String html = "<h1>Управление светодиодом</h1>"; html += "<p><a href=\"/on\">Включить</a></p>"; html += "<p><a href=\"/off\">Выключить</a></p>"; server.send(200, "text/html", html); } void handleOn() { digitalWrite(LED_BUILTIN, LOW); // Включаем светодиод server.send(200, "text/html", "<h1>Светодиод ВКЛЮЧЕН</h1><p><a href=\"/\">Назад</a></p>"); } void handleOff() { digitalWrite(LED_BUILTIN, HIGH); // Выключаем светодиод server.send(200, "text/html", "<h1>Светодиод ВЫКЛЮЧЕН</h1><p><a href=\"/\">Назад</a></p>"); }
✅ Готово! После загрузки кода, откройте в браузере IP-адрес ESP8266, который появится в мониторе порта.

🔬 Практическое задание

  1. Установите Arduino IDE и настройте для ESP8266
  2. Подключите ESP8266 к компьютеру
  3. Замените SSID и пароль в коде на свои
  4. Загрузите код в ESP8266
  5. Откройте IP-адрес в браузере
  6. Управляйте светодиодом через веб-страницу
