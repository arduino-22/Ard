🔌 Урок 7: ESP8266 + Конвертер единиц

← На главную

🎯 Цель урока

Создать веб-сервер на ESP8266, который будет принимать данные с компьютера и конвертировать единицы измерения.

В этом уроке ты соберёшь готовый скетч, загрузишь его в ESP8266 и научишься передавать данные через Wi-Fi.

⚠️ Важно: Для этого урока тебе понадобятся:
  • Плата ESP8266 (NodeMCU, Wemos D1 или аналогичная)
  • USB-кабель для подключения к компьютеру
  • Установленная Arduino IDE (с поддержкой ESP8266)

💻 Скетч для ESP8266

Скопируй этот код в Arduino IDE и загрузи в плату:

/* ============================================================ Урок 7: ESP8266 + Конвертер единиц ============================================================ Этот скетч создаёт веб-сервер, который: 1. Принимает значение, единицу измерения и направление 2. Конвертирует значение по запросу 3. Возвращает результат в формате JSON Подключение: - ESP8266 подключается к Wi-Fi - Открывается веб-сервер на порту 80 - Доступен по IP-адресу, который выводится в монитор порта ============================================================ */ #include <ESP8266WiFi.h> #include <ESP8266WebServer.h> // ===== НАСТРОЙКИ Wi-Fi (ЗАМЕНИ НА СВОИ!) ===== const char* ssid = "Ваш_WiFi_SSID"; const char* password = "Ваш_пароль_WiFi"; // ===== СОЗДАЁМ СЕРВЕР ===== ESP8266WebServer server(80); // ===== КОНВЕРТАЦИЯ ЕДИНИЦ (базовая) ===== float convertUnits(float value, String from, String to) { // Длина if (from == "м" && to == "см") return value * 100; if (from == "см" && to == "м") return value / 100; if (from == "м" && to == "мм") return value * 1000; if (from == "мм" && to == "м") return value / 1000; if (from == "км" && to == "м") return value * 1000; if (from == "м" && to == "км") return value / 1000; // Масса if (from == "кг" && to == "г") return value * 1000; if (from == "г" && to == "кг") return value / 1000; if (from == "т" && to == "кг") return value * 1000; if (from == "кг" && to == "т") return value / 1000; // Если ничего не подошло — возвращаем исходное значение return value; } // ===== ОБРАБОТЧИК ЗАПРОСА /convert ===== void handleConvert() { String valueStr = server.arg("value"); String from = server.arg("from"); String to = server.arg("to"); if (valueStr == "" || from == "" || to == "") { server.send(400, "text/plain", "Ошибка: укажите value, from, to"); return; } float value = valueStr.toFloat(); float result = convertUnits(value, from, to); String json = "{\"result\":" + String(result) + ",\"from\":\"" + from + "\",\"to\":\"" + to + "\"}"; server.send(200, "application/json", json); } // ===== ГЛАВНАЯ СТРАНИЦА (простая) ===== void handleRoot() { String html = "<!DOCTYPE html><html><head><meta charset='UTF-8'><title>ESP8266 Конвертер</title></head><body>"; html += "<h1>ESP8266 Конвертер единиц</h1>"; html += "<p>Пример запроса: <code>/convert?value=5&from=м&to=см</code></p>"; html += "<p>Поддерживаемые единицы: м, см, мм, км, кг, г, т</p>"; html += "</body></html>"; server.send(200, "text/html", html); } // ===== SETUP ===== void setup() { Serial.begin(115200); delay(100); // Подключаемся к Wi-Fi Serial.print("Подключение к Wi-Fi"); WiFi.begin(ssid, password); while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println("\n✅ Подключено!"); Serial.print("📡 IP-адрес: "); Serial.println(WiFi.localIP()); // Настройка маршрутов server.on("/", handleRoot); server.on("/convert", handleConvert); // Запускаем сервер server.begin(); Serial.println("🚀 Сервер запущен!"); } // ===== LOOP ===== void loop() { server.handleClient(); }

📝 Пошаговая инструкция

1 Установи Arduino IDE

Скачай с arduino.cc и установи.

2 Добавь платформу ESP8266

В настройках добавь ссылку:
https://arduino.esp8266.com/stable/package_esp8266com_index.json
Затем установи пакет через менеджер плат.

3 Подключи ESP8266 к компьютеру

Используй USB-кабель. Выбери плату: Инструменты → Плата → NodeMCU 1.0 (или твою модель).

4 Загрузи скетч

Скопируй код выше, вставь в Arduino IDE, замени ssid и password на свои.

Нажми Загрузить (стрелка вправо).

5 Открой монитор порта

Инструменты → Монитор порта. Увидишь IP-адрес ESP8266.

6 Проверь работу

Введи в браузере:
http://[IP-адрес]/convert?value=5&from=м&to=см
Должен прийти JSON-ответ: {"result":500,"from":"м","to":"см"}

🧪 Примеры запросов

  • /convert?value=5&from=м&to=см → 500 (5 метров = 500 см)
  • /convert?value=1000&from=г&to=кг → 1 (1000 г = 1 кг)
  • /convert?value=2&from=км&to=м → 2000 (2 км = 2000 м)

🤖 Дополнительные идеи

  • Добавь больше единиц — расширь функцию convertUnits()
  • Сделай красивую HTML-страницу с формой для ввода
  • Подключи датчик — передавай его показания через сервер
  • Сохраняй историю на SD-карте ESP8266
