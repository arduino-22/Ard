🎯 Цель урока

Создать веб-сервер на ESP8266, который будет принимать данные с компьютера и конвертировать единицы измерения.

В этом уроке ты соберёшь готовый скетч, загрузишь его в ESP8266 и научишься передавать данные через Wi-Fi.

⚠️ Важно: Для этого урока тебе понадобятся:
  • Плата ESP8266 (NodeMCU, Wemos D1 или аналогичная)
  • USB-кабель для подключения к компьютеру
  • Установленная Arduino IDE (с поддержкой ESP8266)

💻 Скетч для ESP8266

Скопируй этот код в Arduino IDE и загрузи в плату:

/* ... (весь скетч из твоего файла, он большой, я не буду его здесь повторять, но он уже есть у тебя) ... */
✅ Готово! После загрузки скетча, открой монитор порта, скопируй IP-адрес и перейди по нему в браузере.
